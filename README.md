Bütçe Takipçisi
Bu proje, Python ile geliştirilmiş basit bir bütçe yönetim uygulamasıdır. Kullanıcıların gelir ve giderlerini kaydetmelerine, mevcut bakiyelerini takip etmelerine ve işlem geçmişlerini görüntülemelerine olanak sağlar.

Özellikler
Gelir ekleme
Gider ekleme
Yetersiz bakiye kontrolü
Tarih ve saat bilgisi ile işlem kaydı
İşlem geçmişini görüntüleme
Verileri dosyaya kaydetme ve program açıldığında geri yükleme
Basit ve kullanıcı dostu terminal arayüzü
Kullanılan Teknolojiler
Python
os modülü
datetime modülü
Dosya işlemleri
Öğrenilen Konular
Bu proje geliştirilirken aşağıdaki Python konuları uygulanmıştır:

Koşul ifadeleri
Döngüler
Hata yakalama
Listeler
Fonksiyonel dosya işlemleri
Tarih ve saat yönetimi
Veri kalıcılığı (persistent storage)
